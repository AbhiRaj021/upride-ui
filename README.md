# UpRide UI 🏍️ A modern, responsive Next.js application for two-wheeler training course selection and booking. Built with TypeScript, Tailwind CSS, and featuring an intuitive user interface for learners to choose their perfect motorcycle training course. ## ✨ Features - **Course Selection Interface**: Interactive UI for choosing between different training courses - **Multi-step Process**: Guided step-by-step course selection process - **Course Options**: - Beginner Blast (10 Sessions to Mastery) - Advanced Ace (05 Sessions to Perfection) - Customizable Course Options - **Pricing System**: Dynamic pricing with GST calculations and discount coupons - **Add-ons**: Additional services like 2W License acquisition - **Responsive Design**: Optimized for both desktop and mobile devices - **Modern UI**: Clean, professional interface with gradient backgrounds and intuitive navigation ## 🛠️ Tech Stack - **Framework**: Next.js 15.3.5 - **Language**: TypeScript - **Styling**: Tailwind CSS v4 - **Icons**: Lucide React - **Font**: Geist (Vercel's font family) ## 🚀 Getting Started ### Prerequisites Make sure you have Node.js installed on your machine. ### Installation 1. Clone the repository: ```bash git clone https://github.com/AbhiRaj021/upride-ui.git cd upride-ui ``` 2. Install dependencies: ```bash npm install # or yarn install # or pnpm install ``` 3. Run the development server: ```bash npm run dev # or yarn dev # or pnpm dev ``` 4. Open [http://localhost:3000](http://localhost:3000) in your browser to see the application. ## 📜 Available Scripts - `npm run dev` - Start development server - `npm run build` - Build the application for production - `npm run start` - Start production server - `npm run lint` - Run ESLint for code quality ## 🏗️ Project Structure ``` src/ ├── app/ │ ├── components/ │ │ └── homepage/ │ │ ├── FirstPage.tsx # Main course selection component │ │ └── HeroSection.tsx # Hero section wrapper │ ├── globals.css # Global styles │ ├── layout.tsx # Root layout │ └── page.tsx # Home page ├── public/ # Static assets └── ...config files ``` ## 🎨 Key Components ### FirstPage Component The main component that renders the course selection interface, featuring: - Step-by-step navigation - Course options with expandable details - Pricing calculator with GST - Coupon application system - Add-on services ### Features Implemented - ✅ Responsive design for mobile and desktop -
